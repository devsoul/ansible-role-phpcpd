PHP Copy/Paste Detector (CPD) Ansible role
=========

If you are a PHP developer and you use PHP Copy/Paste Detector.

Requirements
------------
  - minimum `php  7.1` 

Role Variables
--------------
None

Dependencies
------------
None

Example Playbook
----------------
    - hosts: php
      roles:
        - devsoul.phpcpd

Check if it's installed:
------
    phpcpd --version
